# Программный ассистент для автоматизированного поиска вакансий и интеллектуального улучшения резюме на базе ИИ

## 📚 Описание проекта

Это фронтенд-часть дипломной работы, реализованная на стеке **Next.js 14 + TypeScript + Tailwind CSS + Zustand**.

Проект представляет собой веб-приложение для:

- **Поиска вакансий** с автоматической фильтрацией и ранжированием по релевантности
- **Интеллектуальной адаптации резюме** под конкретные вакансии с помощью ИИ (DeepSeek API)
- **Генерации сопроводительных писем**, оптимизированных для систем ATS

## 🏗️ Архитектура

Проект состоит из трех основных экранов:

1. **Профиль и настройки** — загрузка резюме, настройка параметров поиска
2. **Умная лента вакансий** — отображение вакансий с визуальным индикатором релевантности
3. **Интеллектуальный отклик** — split-экран с генерацией резюме и сопроводительного письма

## 🚀 Установка и запуск

### Требования

- **Node.js** 18.x или выше
- **npm** или **yarn**

### Установка зависимостей

```bash
cd test
npm install
```

### Запуск в режиме разработки

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Сборка для продакшена

```bash
npm run build
npm start
```

## 📦 Моковые данные

Проект включает набор моковых данных в файле `src/lib/mockData.ts`:

- Моковый пользователь (`mockUser`)
- Моковые параметры поиска (`mockSearchPreferences`)
- Моковое базовое резюме (`mockBaseResume`)
- 5 моковых вакансий с разными скорингами (`mockJobs`)
- Моковые сгенерированные артефакты ИИ (`mockGeneratedArtifacts`)

Эти данные позволяют протестировать интерфейс без подключения реального бэкенда.

## 🎯 Основные функции

### Экран "Профиль и настройки"

- Форма загрузки базового резюме (PDF, DOCX, текст)
- Настройка параметров поиска:
  - Грейд (Junior/Middle/Senior)
  - Зарплатная вилка
  - Локация
  - Стек технологий
  - Только удалённая работа

### Экран "Умная лента вакансий"

- Отображение вакансий в виде карточек
- Визуальный индикатор "Скор релевантности" (0-100%)
- Фильтрация по:
  - Минимальному скору релевантности
  - Источнику (hh.ru, Habr, Avito)
  - Удалённой работе
- Сортировка по дате публикации

### Экран "Интеллектуальный отклик"

- Split-экран: слева — вакансия, справа — генерация ИИ
- Кнопка "Сгенерировать резюме под вакансию"
- Лоадер (скелетон) во время генерации
- Результат:
  - Адаптированное резюме с выделенными hard skills
  - Сопроводительное письмо (ATS-оптимизированное)
- Кнопки "Копировать" и "Скачать"

## 📡 API взаимодействия

Бэкенд endpoints (RESTful):

```
GET    /api/jobs              # Получить список вакансий
GET    /api/jobs/:id          # Получить вакансию по ID
POST   /api/jobs/generate     # Сгенерировать резюме (LLM)
GET    /api/user              # Получить профиль пользователя
PUT    /api/user/preferences  # Обновить настройки поиска
POST   /api/user/resume       # Загрузить резюме
```
